# Aplicativo CMS

## Instalação

```bash
composer install
```
<br>

## Banco de dados
#### Para o banco de dados está sendo usado o MySQL, com o script de cricão e inserção dos dados na pasta <i>config/schema</i>.

<br>

## Iniciar servidor

```bash
bin/cake server
```
<br>

## Rota de acesso do CMS
### - Artigos: <i><b> /articles </b></i>
### Exemplo de acesso
```bash
http://localhost:8765/articles
```
